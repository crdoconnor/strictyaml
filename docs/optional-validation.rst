Optional validation
-------------------

Not every key in a YAML mapping will be required. If
you use the "Optional('key')" validator with YAML,
you can signal that a key/value pair is not required.

.. code-block:: yaml
  a: 1
  b: 2

.. code-block:: yaml
  a: 1
  b:
    x: y
    y: z

.. code-block:: yaml
  a: 1

valid_sequence_2.yaml:

.. code-block:: yaml
  a: 1

.. code-block:: yaml
